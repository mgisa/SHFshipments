 ## 📊 Dashboard Story

The SFH Health Logistics Monitoring Dashboard analyses 10,324 USAID SCMS shipment records spanning 2006–2015 across 43 countries to surface a single strategic insight: Air freight costs between **4–5× more per kilogram** than Truck, yet delivers **no measurable improvement in on-time performance**, overall on-time rates hover around the same level regardless of shipment mode.

The dashboard walks through this story in seven tabs: an **Overview** of shipment volumes, mode share, and on-time trends; a **Supply Chain Map** showing country-level freight costs and delivery reliability; an **Efficiency** tab decomposing late versus on-time variance across all years; a **Director's Story** tab that builds the cost-premium case with KPI cards and a 10-year cost-per-kg time series across Air, Truck, and Ocean modes; an **EDA Report** tab documenting the three data quality issues found in the raw dataset (non-numeric freight costs, extreme outliers, and the string "N/A" sentinel in shipment mode); a filterable **Data Explorer** with CSV and Excel
export; and a **Downloads** tab for the full cleaned dataset and HTML report.

The core recommendation to programme directors: adopt a mode-switching policy for bulk, non-urgent commodities — ARVs and routine HRDT kits on Truck-feasible routes — to reduce freight unit cost substantially without sacrificing delivery reliability.

## 🗃️ Dataset

**USAID SCMS Delivery History Dataset**
- 10,324 rows × 33 columns | Period: 2006 – 2015
- Source: [USAID Supply Chain Management System](https://data.usaid.gov)
- File to place in `data/raw/`: `SCMS_Delivery_History_Dataset.csv`
- Encoding: latin-1 (required for country names with accents e.g. Côte d'Ivoire)

**Key data quality issues found in raw data:**
1. ~40% of `Freight Cost (USD)` values are non-numeric text strings requiring 3-category treatment
2. Extreme right skew in numeric freight values — outliers exceed $500k per shipment
3. String `"N/A"` used as a sentinel in `Shipment Mode` — invisible to `is.na()`
